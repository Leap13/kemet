== Description ==

Kemet is a Fast, Reliable, and World-Class Full Site Editing WordPress Block Theme suitable for business websites, blogs, personal portfolios. Kemet is bundled with outstanding pre-built Templates, Template Parts, and designer-made Block Patterns to help you get your WordPress website almost with no effort.
